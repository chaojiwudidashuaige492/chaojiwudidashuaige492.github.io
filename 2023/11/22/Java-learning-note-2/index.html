2.变量使用

  1. 声明变量 int a;
  2. 赋值 a=60;
  3. 使用
    或者在声明的时候进行赋值（初始化变量）
    int a=60;

基本数据类型

  1. 整数类型
  2. 小数（浮点）类型
  3. 字符型_char[2]
  4. 布尔型_boolean[1]

  1. 类(class)
  2. 接口(interface)

基本数据类型汇总 : [byte,short,int,long,float,double]char,boolean

整型

–整型种类
byte型:1字节byte n1=10;
short型:2字节short n2=10;
int型:4字节int n3=10;
long型:8字节long n4=10;
都留有一个符号位（第一位）决定正负性

–整型的使用细节

  1. Java的整型常量默认为int型，声明long型常量须后加’L’或’l’
    int n1=10L编译报错，因为10L为long型，n1为int型

浮点数

–浮点型种类
float型:4字节float n1=3.11;单精度
double型:8字节double n2=3.11;双精度
浮点数=符号位+指数位+尾数位
尾数部分可能丢失，造成精度损失（小数都是近似值）

–浮点型使用细节

  1. Java浮点型常量默认为double型，声明float类型的常量，须后加’f’或’F’
    float n1 = 1.1出现错误，n1为float型，而1.1默认为double型常量（会造成精度丢失）
    应为float n1=1.1f
  2. 浮点型常量的科学计数法形式:
    5.12e2=5.12*10^2----5.12E-2=5.12*10^-2
  3. 浮点数使用陷阱：不要对运算结果为小数的进行相等判断（2.7！=8.1/3）
    如果是直接查询得到的小数或是直接赋值的小数，是可以直接判断相等的
    （正确的使用方法） ：n1==n2------>Math.abs(n1-n2)<精度a既n1-n2的绝对值在某个精度范围内判断

字符型

1.表示单个字符

2.两个字节（可以存放汉字）

3.字符类型可以直接存放一个数字（输出数字指代的字符）

4.字符类型使用细节

  1. 字符常量是用单引号(‘’)括起来的单个字符。
  2. java中允许使用转义字符’\‘来将其后的字符转变为特殊字符型常量(‘\t’是一个字符
  3. )
  4. char类型的本质是一个整数，可以直接给char类型赋值一个整数，然后输出时，是Unicode码对应的字符
    （要输出对应的数字时，可以使用int进行一个类型转换）
  5. char类型是可以进行运算的，相当于一个整数
    补充：System.out.println('a'+10);在进行”+”运算时进行了类型升级，’a’由char型升级成为int型进行”+”运算，最后的结果也为int型

5.字符型本质探讨

  1. 字符型存储到计算机中，需要将字符型对应的码值(整数)找出来。例如
    存储: ‘a’–>码值97–>二进制–>存储
    读取：二进制–>码值97–>’a’–>显示

  2. 字符编码表（了解）{
    ASCII(ASCII编码表，一个字节表示，一共128个字符)

    Unicode(Unicode编码表，固定大小的编码，使用两个字节来表示字符
    ，字母和汉字统一都是占用两个字节，这样浪费空间)Unicode码兼容ASCII码

    utf-8(utf-8编码表，大小可变的编码，字母使用一个字节，汉字使用3个字节)可以理解为对Unicode的改进

    gbk(可以表示汉字，而且范围广，字母使用1个字节，汉字2个字节)

    gb2312(可以表示汉字，gb2312<gbk)

    big5 码(繁体中文)

}

04_基本数据类型的转换

自动类型转换

  1. 当Java程序在进行赋值或者运算时，精度小的类型自动转换为精度大的数据类型，这个就是自动类型转换
  2. 数据类型排序,按精度(容量)排序
    char-->int-->long-->float-->double
    byte-->short-->int-->long-->float-->double

自动类型转换注意和细节

  1. 有多种数据类型的混合运算时，系统首先自动将所有数据转换成容量最大的那种数据类型，然后再进行计算
  2. 当我们把精度(容量)大的数据类型赋值给精度(容量)小的数据类型时，就会报错。反之将精度(容量)小的数据类型赋值给精度(容量)大的变量时，就会进行自动类型转换。
  3. byte，short和char之间，不会相互自动转换
  4. 将整型常量赋值给byte或short类型时，系统会先判断该类型能否存储该常量。如果是将变量赋值给该变量，就会直接判断类型
  5. byte,short,char类型三者可以计算，在计算时首先转换为int类型，结果为int类型
  6. boolean不参与转换
  7. 自动提升原则：表达式结果的类型自动提升为，操作数中最大的类型

强制类型转换

自动类型转换的逆过程，将容量大的数据类型转换为容量小的数据类型。使用时要加上强制转换符()，但可能造成精度降低或溢出，格外要注意

强制类型转换细节说明

  1. 当进行数据从大–>小，就要用到强制类型转换
  2. 强制类型转换符号”()”只针对于最近的操作数有效，往往会使用小括号提升优先级
  3. char类型可以保存int的常量值，但不能保存int的变量值，需要进行强制类型转换
  4. byte和short，char类型在进行运算时，当作int类型处理

基本数据类型和String类型的转换

  1. 基本数据类型转String类型:将基本类型的值+””
    “+”字符串加其他 = 字符串 Char型可以变换成String型
  2. String类型转基本数据类型:通过基本类型的包装类,调用parseXX方法(parse=解析) 把字符串String转成字符char—>得到字符串的第几个字符
  3. 注意事项:
    （1）：在将String类型转成基本数据类型时，要确保String类型能够转成有效的数据。例如:
    int n1 = Integer.parseInt("hello");—报错
    （2）：如果格式不正确，就会抛出异常，程序就会终止（没能转成有效的数据）

附加内容：键盘输入语句

介绍：需要接收用户输入的数据，使用键盘输入语句。需要一个扫描器（对象）， 就是Scanner

步骤

  1. 导入该类的所在包中的类。java.util
    import java.util.Scanner;
  2. 创建该类的对象（声明变量）
    Scanner scanner = new Scanner(System.in);
  3. 调用里面的功能(接受用户输入)
    String name = scanner.next();接收一个字符串
    int age = scanner.nextInt();接收一个int
    double salary = scanner.nextDouble();接收一个double
