Chapter_03运算符

(1). 算术运算符

1. 一元运算符

  1. 正号”+”: +7—>7
  2. 负号”-“: b=11,-b—>-11

2. 二元运算符：

  1. 加”+”: 9+9—>18
  2. 减”-“: 10-8—>2
  3. 乘”“: 78—>56
  4. 除”/“: 9/9—>1
  5. 取余”%” 11%9–>1可以用于浮点数（浮点数计算公式 : a%b = a - (int)a/b * b）

3. 自增/减运算符

  1. 自增(前):先加上，再取值
  2. 自减(前):先减去，再取值
  3. 自增(后):先取值，后加上
  4. 自减(后):先取值，后减去

4. 字符串相加:”+”—-拼接字符串

5.细节说明

  1. “\“除法的使用；
    （1）整数/整数=整数，小数部分会被舍去
    （2）a/b，其中任意一个为浮点数，结果都为浮点数
    double n1 = 10/4 n1 = 2.0(先计算右边再赋值给左边)
  2. “%”取余运算的使用：
    （1）关键公式：a % b = a - a / b * b
  3. 自增/自减 (–同理)

算术运算符面试题:

1.

（1）：先将右边的i的值放入栈中（2）：右边的i进行自增（3）：将栈中的i的值赋给左边的i
表现在j=i++中极为把i自增前的值赋值给j

原理如下（1）：右边的i进行自增（2）：将右边的i的值放入栈中（3）：将栈中的值赋给左边

(2).关系运算符(比较运算符)

介绍：关系运算符的结果都是boolean型，要么是true要么是false

总览：

  1. == 相等于
  2. != 不等于
  3. < 小于

  4. 大于

  5. <= 小于等于

  6. = 大于等于

  7. instanceof 检查是否是类的对象
    "lcx" instanceof String --->true(instance=实例)

(3).逻辑运算符

介绍：用于连接多个条件(多个关系表达式)，最终的结果也是一个boolean值

总览：

  1. & 逻辑与–都为true才是true
  2. | 逻辑或 –都为false才是false
  3. ^ 逻辑异或–相同为false，不同为true
  4. && 短路与–都为true才是true
  5. || 短路或–都为false才是false
  6. ! 取反–true变false，false变true

细节：

  1. 逻辑与和短路与：
    （1）：逻辑与：先算出运算符两边的bool值，再得出结果
    （2）：短路与：先算出左边的bool值，如果为false那么跳过运行右边的表达式，直接得出false的结果（效率高）
  2. 逻辑或和短路或：
    （1）：逻辑或：先算出运算符两边的bool值，再得出结果
    （2）：短路或：先算出左边的bool值，如果为true那么跳过运行右边的表达式，直接得出true的结果（效率高）

(4).赋值运算符

介绍：赋值运算符就是将某个运算后的值，赋给指定的变量

总览：

  1. += a+=b–a=a=b
  2. -= a-=b–a=a-b
  3. = a=b–a=a*b
  4. /= a/=b–a=a/b
  5. %= a%=b–a=a%b

细节：

  1. 先计算右边，再赋值给左边
  2. 赋值运算符的左边只能是变量
  3. 复合赋值运算符会进行类型转换
    byte b=1;b+=3;b仍然为byte类型
    即为b = (byte)(b + 3);
    而直接写b = b+3则会报错
  4. 自增自减运算符同样满足类型转换，即
    b++,b=(byte)(b+1)

(5).三元运算符（条件判断运算符）

基本语法:

条件表达式?表达式1:表达式2;

如果表达式的值为true，运算后的结果为表达式1.（只对表达式1进行判断，并返回表达式1的值）

如果表达式的值为false，运算后的结果为表达式2.（只对表达式2进行判断，并返回表达式2的值）

使用细节

表达式1和表达式2要为可以赋给接受变量的类型（或者可以通过自动转换赋值给变量的类型）

(6).位运算符（符号）

总览：

  1. & 按位与 ：两位全为1，结果为1，否则为0
  2. | 按位或 ：两位有一个为1，结果为1，否则为0
  3. ^ 按位异或 ： 两位一个为1一个为0，结果为1，否则为0
  4. ~ 按位取反 ： 0–>1,1–>0
  5. >> 算术右移 ：低位溢出，符号位不变，并用符号位补齐空余的高位
    换算成10进制相当于除2
  6. << 算术左移 ：符号位不变，低位补0
    换算成10进制相当于乘2
  7. >>> 逻辑右移（无符号右移） ： 低位溢出，高位补0（区别于算术右移）（没有<<<符号）

细节：

  1. >>与<<：int a=1>>2;表示算术右移两位
    int c=1<<2;表示算术左移两位

(7).位运算符（进制）

进制总览：

  1. 二进制：0，1.满2进1，以0b或0B开头(binary)
  2. 十进制：0-9.满10进1
  3. 八进制：0-7.满8进1，以数字0开头
  4. 十六进制：0-9以及A(10)-F(15).满16进1，以0x或0X开头，此处的A-F不区分大小写(hexadecimal)

进制转换(十进制转其他进制)：

原理
(以34为例)：
17(被2除1次) 0(被2除0次)//
8(被2除2次) 1(被2除1次)//
4(被2除3次) 0(被2除2次)//
2(被2除4次) 0(被2除3次)//
1(被2除5次) 0(被2除4次)//
0(被2除6次) 1(被2除5次).
结果为：100010—->1 * 2^5 + 1 * 2^1

  1. 十进制转换成二进制：将该数不断除2，直到商为0，然后将每步得到的余数倒过来
  2. 十进制转换成八进制：将该数不断除8，直到商为0，然后将每步得到的余数倒过来
  3. 十进制转换成十六进制：将该数不断除16，直到商为0，然后将每步得到的余数倒过来

进制转换(二进制转其他进制)

  1. 二进制转八进制：从低位开始，将二进制每三位结合成一组，转成对应的八进制数(2^3=8)
  2. 二进制转十六进制：从低位开始，将二进制每四位结合成一组，转成对应的十六进制数(2^4=16)

进制转换(其他进制转成二进制)

  1. 八进制转换成二进制：从低位开始，将八进制数的每一位，转成对应的一个3位的二进制数
  2. 十六进制转换成二进制：从低位开始，将十六进制数的每一位，转成对应的一个4位的二进制数

(8).位运算符(原码，反码，补码)

  1. 二进制的最高位是符号位:0表示正数，1表示负数
  2. 正数的原码，反码，补码都一样
  3. 负数的反码=它的原码符号位不变，其他位取反(0–>1,1–>0)
  4. 负数的补码=它的反码+1//同理//负数的反码=它的补码-1
  5. 0的反码，补码都是0
  6. java没有无符号数，Java中的数都是有符号的
  7. 在计算机运算的时候，都是以补码的方式来运算的（无论什么运算都是）（且补码运算得到的结果也为补码，要转回原码才能得出准确答案）
  8. 当我们看运算结果的时候，要看他的原码
    举例（1）：2&3 举例（2）~-2 举例（3）~2
